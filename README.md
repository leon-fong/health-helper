## 功能

- 记录饮食（早中晚）、运动的数据
- 自动生成健康报告/分析
- 个性化推荐食谱
- 智能计算推荐摄入蛋白质
- 食物多项数据含量对比
- 同步微信步数

## 使用

1. 克隆或下载此项目
2. 使用 vscode 打开该项目
3. 安装并使用 `pnpm` 安装依赖
4. 在项目根目录运行 `pnpm dev:weapp` 启动本地服务
5. 在微信开发者工具导入 `dist` 目录下的小程序代码
