## 使用
- 克隆或下载此项目
- 使用 [Visual Studio Code](https://code.visualstudio.com/) 打开该项目
- 安装并使用 [pnpm](https://pnpm.io/) 安装依赖
- 在项目根目录运行 `pnpm dev` 启动本地服务
- 在微信开发者工具导入 `dist` 目录下的小程序代码
